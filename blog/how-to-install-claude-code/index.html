如何安装 Claude Code，只需两步

2026年5月10日

如果你到现在还没有用上claude code 或者 openclaw，又或者 Hermes-agents

那么我能想象你一定在安装 agent过程中遇到了种种障碍，各种奇奇怪怪的问题层出不穷，即使你有豆包、千问、元宝这样的顶级老师在。依然没有解决问题。

其实更可怕的阻力是自己的观念和思维限制——我不懂代码、不懂高科技的东西。

我呢，并不会写代码。曾经自学 Python 和计算机半途而废。但，在这一次 AI 中，我没有一点的恐惧，因为它对于我来说并不是完全的陌生。我不会写代码，但我稍微会读一读。我的英语并不好，但并不妨碍我阅读英语书籍和博客。

所以，AI 也是如此。

最近，我发现一个对于小白（那种完全不懂的小白）非常友好的安装claude code 或者 openclaw，又或者 Hermes-agents 的方式。简单到只有两步：

一：从网上下载腾讯的qclaw，这是一个套壳版的 openclaw，而且大厂送你”几乎”用不完的 token。

二：告诉qclaw，帮我安装claude code、Hermes-agents

它就会给你安装了，有什么问题，它自己摸索就解决了。

你的事情可能是给权限，登录的事情。搞定。
